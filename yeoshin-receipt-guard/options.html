v1.0.0 — 영수증 자동 검증 설정

공공데이터포털(data.go.kr)에서 국세청_사업자등록정보 진위확인 및 상태조회 서비스 API를 신청 후 발급받은 인증키를 입력하세요.

영수증 OCR에 Gemini Vision API를 사용합니다. 미설정 시 사업자번호를 인식할 수 없습니다.
모델: gemini-2.0-flash-lite (가성비 최적) — API Key 발급

Supabase 프로젝트 Settings → API → Project URL을 복사해 입력하세요.
미입력 시 중복 해시를 이 브라우저 로컬에만 저장합니다.

Supabase 프로젝트 Settings → API → anon public 키를 입력하세요.

API Key 발급 방법

  1. 공공데이터포털 (data.go.kr) 회원가입/로그인
  2. "국세청_사업자등록정보 진위확인" 검색
  3. "활용신청" 버튼 클릭 후 신청
  4. 승인 후 마이페이지 → 개발계정 → 인증키 복사
  5. 위 입력란에 붙여넣기 후 저장
